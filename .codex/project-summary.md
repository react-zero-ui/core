# Project Summary

## Overview

- React Zero-UI is a React UI state library plus CLI for build-time, CSS-driven UI state.
- It serves React developers using Next.js or Vite who want faster UI updates without typical re-render-heavy state patterns.

## What It Does

- Lets apps drive global and scoped UI state through `data-*` attributes and CSS variables instead of React re-renders.
- Ships a PostCSS integration, a Vite integration, experimental SSR-safe interaction helpers, and a `create-zero-ui` CLI for project setup.

## Scope

- This repo is a monorepo for the published library `@react-zero-ui/core`, the `create-zero-ui` CLI, docs, tests, and example/demo code.
- Current scope is focused on a stable core library, better docs, compatibility with modern Next.js/Vite workflows, and contributor-friendly open source maintenance.
