Zero-UI is a global UI state engine that updates your interface using `data-*` attributes â€” **no React state, no VDOM, no re-renders**.

It works by:

1. Scanning your components for `useUI()` hooks

3. Updating `data-*` attributes on `<body>` to drive the UI instantly
